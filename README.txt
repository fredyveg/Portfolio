🖥️ Mi Portafolio Personal

Este es mi portafolio personal, desarrollado con HTML, CSS y JavaScript, enfocado en mostrar mis habilidades, proyectos y experiencia profesional de manera atractiva y accesible. Además, la página es completamente responsive, adaptándose a diferentes tamaños de pantalla para mejorar la experiencia del usuario.

✨ Características

- 📱 Diseño responsive: Optimizado para dispositivos móviles, tablets y computadoras de escritorio, con ajustes de visualización específicos para varios tamaños de pantalla.
- 📑 Menú lateral animado: En dispositivos pequeños, el menú de navegación se transforma en un menú lateral desplegable, con una animación suave al abrirse y cerrarse.
- 🎨 Efectos de transición: Elementos interactivos y botones tienen efectos de transición para una experiencia visual más dinámica.

🛠️ Tecnologías Utilizadas

- HTML5: Estructura de la página.
- CSS3: Estilos visuales, diseño responsive y animaciones.
- JavaScript: Interactividad, control del menú lateral y efectos en los elementos.
